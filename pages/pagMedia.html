Imagem dinâmica

A tag <picture> nos permite trabalhar com imagens de formas variadas. Hoje aprenderemos a usar a tag <source> dentro da tag <picture>, ela nos permite manipular as imagens dependendo do tamanho do navegador aberto.

Reproduzindo aúdio

Aqui aprenderemos a utilizar a tag <audio> para inserir um arquivo de aúdio na página.

Controle de formatos

Para esse aqui não teremos como testar, mas fica a dica para driblar falhas de execução de aúdio. Apenas visível no back-end! Pois mostra a utilização novamente da tag <source> só que dessa vez para aúdio.

Utilizando vídeos

Vídeos internos

Aprenderemos aqui a utilizar a tag <video> para inserir vídeos em nossa página HTML.

Controle de formatos

Da mesma forma que foram feitos com o aúdio e a imagem, também podemos utilizar a tag <source> nos vídeos. Apenas será visível no back-end!

Vídeos externos

Aprenderemos a inserir vídeos de sites externos, para uma melhor performance e custeio de hospedagem. Para o exemplo utilizaremos vídeos do YouTube, pois o site já compartilha um código pronto de forma prática para facilitar nossa vida.

Outra opção padrão é o site Vimeo.
